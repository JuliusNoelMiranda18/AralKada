## 🧐 What is AralKada?

**AralKada** (from Filipino *"aral"* = learn + *"kada"* = every/everyone) is an AI-powered, all-in-one academic companion built specifically for **Filipino senior high school and incoming college students**. It consolidates entrance-exam preparation, AI-driven study tools, Philippine dialect learning, scholarship discovery, college application tracking, and commute planning into a single, cohesive application.

## ✨ Features

### 🧠 Study Buddy (AI Study Guide Generator)
Upload your documents (PDF, PPT, PPTX) or provide free-text prompts to generate comprehensive study materials.
- **Teacher Mode:** Tailors content based on learning styles (Visual, Auditory, Reading/Writing).
- **Auto-generated Diagrams:** Mermaid.js diagrams for visual learners.
- **Interactive Quizzes & Flashcards** and "Explain Like I'm 5" summaries.
- **AI Chat Companion ("Cardo")** grounded in your uploaded document context.

### 📝 Mock Exam Engine
Timed, multiple-choice exam simulator using real past entrance exam questions.
- **5 Exam Families Supported:** UPCAT, USTET, ACET, PUPCET, DCAT.
- **AI Integration:** Hint generation and similar-question generation in real-time.
- **Extensive Database:** 26 exam registry entries with 7 processed JSON datasets ready for simulation.

### 🗣️ Dialect Learning Module
Duolingo-style Philippine dialect learning powered by AI.
- **Immersive Curriculum:** 6 structured units covering 25 unique practical scenarios (ordering street food, riding jeepneys, etc.).
- **Dynamic Content:** Real-time phrase translations, hints, and explanations generated by AI.

### 🎓 College Application Hub
A comprehensive university browser and application tracker.
- **School Finder:** Location-based filtering using OpenStreetMap and AI matching.
- **Calendar Dashboard:** Track application deadlines, exam schedules, and results across 25 indexed universities (State, Private, LUC).

### 💰 Scholarship Feed
Curated and AI-augmented Philippine scholarship directory.
- **Extensive Coverage:** Government (DOST, CHED), Corporate (SM, Aboitiz), and Private scholarships.
- **AI Search:** Degree-specific scholarship matching using Groq.

### 📓 Smart Note-Taking (Tala)
A fully-featured, GoodNotes-inspired workspace.
- **4 Editor Types:** Document (Rich Text via TipTap), PDF Annotator, Whiteboard (Fabric.js), and Notebook.
- **AI Note Formatting:** Automatically formats notes into Cornell, Outline, or Flashcard styles.

### 🚍 Commuter Guide
AI-generated public transit routing with safety and cost optimization.
- **Multiple Transport Modes:** Jeepney, Bus, LRT, MRT, UV Express, Tricycle, Ferry, and Walking.
- **Interactive Maps:** Leaflet + OpenStreetMap integration.

## 🚀 Getting Started

### Prerequisites

- **Node.js** 18+
- **npm** or **yarn**
- A **Groq API key** ([get one here](https://console.groq.com))

### Installation

```bash
# Clone the repository
git clone https://github.com/JuliusNoelMiranda18/AralKada.git
cd AralKada

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
```

Add your API keys to the `.env.local` file:

```env
GROQ_API_KEY=your_groq_api_key_here
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

- 🇵🇭 **Entrance Exams:** Tailored mock exams for UPCAT, USTET, ACET, PUPCET, and DCAT.

- 🚍 **Local Transit:** Commute routing explicitly including Tricycles, Jeepneys, and UV Express.

## 🤖 AI Architecture

AralKada utilizes **Groq Cloud** for ultra-fast, structured inference across all platform features.

- **Model:** `llama-3.3-70b-versatile` globally
- **Structured Outputs:** Uses `json_object` format to guarantee reliable rendering in lessons and exams.
- **Dynamic Temperature Scaling:** 
  - `0.2` for factual transit routing
  - `0.3` for study guide generation
  - `1.0` for diverse dialect scenario generation.

---

## 🔒 Privacy

AralKada follows a **zero-database, client-first architecture**:

- **No User Accounts:** Frictionless usage with zero sign-up requirements.
- **Client-Side Storage:** All notes, preferences, mock exam progress, and bookmarks are stored securely on your browser using `localStorage`.
- **Ephemeral AI:** API calls to Groq are stateless. File uploads and AI context are processed temporarily and discarded.
